**3rd semester  mini-project**

Scope: 
  Threadit is a simplified version of Reddit where users can post, vote, and comment on topics.
  Leveraging ASP.NET Web API for backend logic and Blazor WebAssembly for the frontend interface.

Technologies:
  ASP.NET Web API
  Blazor WebAssembly
  SQLite 
  C#
  Visual Studio/Rider 
  GitHub
